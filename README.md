DevTinder is a developer-focused matchmaking platform inspired by Tinder.

It allows developers to connect, collaborate, and build professional relationships.

Users can swipe right/left based on interest and create meaningful tech connections.

Teck-Stack Used :
Frontend - React.js // Tailwind Css

Backend -Node.js // Express.js 

Database - MongoDB

Other Tools :
JWT Authentication
Socket.io (for chat, if used)

Features :
User Authentication(login /signUp )
Edit User Profile
Sent Request to another User(interested / NotInterested )
Chat Features
Tech Stack Based Matching 

“This project demonstrates full-stack development skills including authentication, real-time interaction, and scalable backend design.”
